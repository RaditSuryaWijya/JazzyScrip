# Project Brief

## Informasi Dasar
- **Nama Proyek**: Tugas_Kuliah
- **Tipe**: Roblox Lua Scripting Project
- **Lisensi**: MIT License
- **Copyright**: © 2025 akmilia
- **Bahasa**: Lua (Roblox)

## Deskripsi Proyek
Proyek ini adalah kumpulan script Roblox Lua untuk automation game yang mencakup berbagai fitur seperti:
- Auto Fishing (berbagai mode: Blatant, Fast, Perfect)
- Auto Quest System
- Auto Sell/Buy System
- Teleport System
- GUI Components
- Security Loader dengan enkripsi
- Memory Leak Fixes
- Performance Optimizations

## Struktur Proyek
```
Tugas_Kuliah-main/
├── SecurityLoader.lua          # Loader utama dengan enkripsi dan security
├── Project_code/               # Kode utama terorganisir
│   ├── BlatantV2.lua          # Auto fishing mode Blatant V2
│   ├── Utama/                 # Modul utama (fishing, movement, dll)
│   ├── Quest/                 # Auto quest modules
│   ├── ShopFeatures/          # Auto sell/buy features
│   ├── TeleportSystem/        # Teleport modules
│   ├── Camera View/           # Camera modules
│   └── Misc/                  # Utility modules
└── [various GUI and test files]
```

## Tujuan Proyek
1. Menyediakan sistem automation yang komprehensif untuk game Roblox
2. Mengimplementasikan security measures untuk melindungi script
3. Mengoptimalkan performa dengan fix memory leaks
4. Menyediakan GUI yang user-friendly untuk kontrol fitur
5. Mendukung multiple modes untuk berbagai use cases

## Scope Proyek
- **In Scope**: 
  - Auto fishing dengan berbagai mode
  - Auto quest automation
  - Shop automation (sell/buy)
  - Teleport system
  - GUI interface
  - Security & encryption
  - Performance optimization
  
- **Out of Scope**:
  - Tidak termasuk game hacking yang melanggar ToS
  - Tidak termasuk exploit tools

## Target Platform
- Roblox Platform
- Executor: Jazzy (dari konteks SecurityLoader)
- Compatible dengan berbagai Roblox executors

## Status Proyek
Proyek dalam tahap development aktif dengan berbagai versi dan iterasi fitur.
